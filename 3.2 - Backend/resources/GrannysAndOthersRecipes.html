get /recipes/search/query/{searchQuery}/amount/{number}

search for recipe in the spooncular API, using a search query. will return results according to the {num} param. Can filter results by Cuisine, Diet, Intolerance.

Path parameters

searchQuery (required)
Path Parameter — The search query for searching a query in the API
number (required)
Path Parameter — number of results to retrieve - 5/10/15

Query parameters

cuisine (optional)
Query Parameter —
diet (optional)
Query Parameter —
intolerances (optional)
Query Parameter —

Return type

map[recipePreview]

Example data

Content-Type: application/json
{
  "1" : {
    "image" : "google.com/amazingHamburger111",
    "title" : "American Hamburger",
    "readyInMinutes" : 60,
    "aggregateLikes" : 100,
    "vegan" : false,
    "vegetarian" : false,
    "glutenFree" : false
  }
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Successful operation - search results returned

404

No recipes found for the search param

422

number must be 5/10/15 OR searchQuery must contain only letters OR number must be an integer number OR Inserted invalid option for cuisine OR Inserted invalid option for diet OR Inserted invalid option for intolerances.
